### how to install?
 1. git clone https://github.com/syolin/shit-algorithm.git
 2. cd shit-algorithm
 3. npm install
 4. npm install nodemon --save

### how to use?
 npm start
 ( node index.js )
 npm run dev
 ( nodemon index.js )
